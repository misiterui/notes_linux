# 快捷键

| 按键     | 作用                         |
| ------ | -------------------------- |
| ctrl+G | 获取线上说明(help)               |
| ctrl+X | 退出nano软件，会提示是否保存           |
| ctrl+O | 保存                         |
| ctrl+R | 从其他文件读入数据，可以将某个文件的内容贴在本文件中 |
| ctrl+W | 搜索字符串                      |
| ctrl+C | 说明目前光标所在处的行数与列数等信息         |
| ctrl+_ | 输入行号，让光标快速移动到该行            |
| alt+Y  | 校正语法功能，打开/关闭               |
| alt+M  | 支持鼠标来移动光标的功能               |
