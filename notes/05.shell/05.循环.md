> 三种循环

## 1. for循环

##### 1. 遍历列表
```bash
#!/bin/bash

for item in apple banana cherry
do
	echo "Fruit: $item"
done
```

##### 2. 遍历数字范围
```bash
#!/bin/bash

for i in {1..5}
do
	echo "Number: $i"
done
```

##### 3. 使用seq命令
```bash
#!/bin/bash

for i in $(seq 1 5)
do
	echo "Seq: $i"
done
```

##### 4. C语言风格
```bash
#!/bin/bash

for ((i=0; i<5; i++))
do
	echo "C-style: $i"
done
```

##### 5. 遍历文件
```bash
#!/bin/bash

for file in *.txt
do
	echo "File: $file"
done
```

##### 6. 遍历文件内容
```bash
#!/bin/bash

for user in $(cat users.txt)
do
	echo "User: $user"
done
```

## 2. while循环
##### 1. 计数器方式
```bash
#!/bin/bash

count=1
while [ $count -le 5 ] # `-le` 是条件测试中的一个操作符，表示“小于或等于”（less than or equal to）
do
	echo "Count: $count"
	count=$((count+1))
done
```

##### 2. 读取文件行
```bash
#!/bin/bash

while read line
do
	echo "Line: $line"
done < input.txt
```

##### 3. 无限循环（需要break退出）
```bash
#!/bin/bash

while true
do
	echo "Press Ctrl+C to stop"
	sleep 1
done
```

##### 4. 条件控制
```bash
#!/bin/bash

password=""
while [ "$password" != "secret" ]
do
	read -p "Enter password: " password
done
echo "Access granted!"
```

##### 5. 使用命令返回值
```bash
#!/bin/bash

while ping -c 1 google.com &> /dev/null
do
	echo "Network is up"
	sleep 5
done
echo "Network is down"
```

## 3. until循环
##### 1. 与while相反，条件为假时执行
```bash
#!/bin/bash

count=1
until [ $count gt 5 ]
do
	echo "Count: $count"
	count=$((count+1))
done
```

##### 2. 等待服务启动
```bash
#!/bin/bash

until curl -s http://localhost:8080 > /dev/null
do
	echo "Waiting for server..."
	sleep 2
done
echo "Server is up!"
```

##### 3. 文件存在检查
```bash
#!/bin/bash

until [ -f /tmp/ready.flag ]
do
	echo "Waiting for file..."
	sleep 1
done
echo "File found!"
```
